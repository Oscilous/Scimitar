# Scimitar V2 (open-source airframe)

This repository hosts the design files and docs for Scimitar V2.

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18325531.svg)](https://doi.org/10.5281/zenodo.18325531)

## Tech specs

- **Wingspan:** ~80 cm; **length:** ~41 cm.
- **Airframe weight:** ~395 g (bare + 2 wing spars).
- **Typical build (no battery):** ~540 g (FC, ESC, RX, 2× servos, antennas, HD cam + VTX).
- **UAV example weight:** ~700 g with a 4S 1300 mAh (≈160 g) pack.
- **Recommended motor class:** 2204–2206 / 2306 / F60; **prop:** 6×4″–6×5″.
- **Recommended battery:** 4S, ~150–230 g pack.

## Contents
- **Hardware license:** CERN-OHL-P v2.0 (permissive)
- **Docs license:** CC BY 4.0 (for text/images)
- **Design files:** native Fusion 360 (.F3Z with components) + STEP; 3MF for printing
- **BoM:** minimal materials/parts list
- **Printing & assembly:** see `/docs`

## Quick start
1. Download `/hardware/meshes/` for printing or open `/hardware/native/` (.F3Z) or `/hardware/native/step-exports/` (.STEP) to edit.
2. See `/docs/printing.md` for slicer settings (nozzle, layer height, infill, walls).
3. See `/docs/assembly.md` for join order, glue and general assembly.
